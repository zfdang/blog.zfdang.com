从BTT到HammerSpoon

之前一直都用BetterTouchTool来做一些快捷的操作。但是这玩意是收费的，之前买的一个license, 现在居然失效了…

就想尝试着看看有没有类似的软件可以替代，于是找到了HammerSpoon!

HammerSpoon安装完之后，什么也干不了，需要自己来写配置文件，从而实现各种快捷操作，这个导致了它比较高的使用门槛！
