使用 eSIM 的方法总结

使用eSIM的最简单的方法，就是你的手机直接支持eSIM。

如果你的手机没有eSIM的硬件支持，但是又想使用eSIM的号码，那么使用支持写入eSIM号码的实体SIM卡，就是唯一的选择了。

当前民间与开源社区形成了多种可行方案，大体可分为以下几类。

一、技术路线分类

1. 开源技术路线：基于 open eUICC

代表方案：OpenEUICC / 9eSIM / 单芯片方案

特点：

  • 以开源的 open eUICC 项目为基础。
  • 多采用 单芯片 eSIM 白卡。
  • 安卓手机可直接写卡。
  • 苹果 iPhone 需借助 NFC 读卡器写卡（因 iOS 限制）。

2. 自研软件方案

代表：5Ber、Xesim、Plan B (Switch)、eSIM.me

特点：

  • 技术核心和实现方式与开源路线类似。
  • 依旧主要使用 单芯片 eSIM。
  • 写卡模式相同：
    • 安卓手机可直接写卡
    • 苹果手机通过切卡/写卡工具完成

这些方案一般使用自研 App 对 eSIM 卡进行管理和写入。

3. 自研硬件 + 软件方案

代表：ESTK 方案

特点：

  • 使用 双芯片 eSIM 白卡（安全性更高、功能更完整）。
  • 配合自研硬件与软件体系。
  • 产品迭代速度快。
  • 对兼容性、稳定性有优势。

4. 其他白牌 eSIM 卡

  • 市面上的多数白牌 eSIM 卡仍以 单芯片方案 为主。
  • 成本较低，功能较基础
  • 需要搭配写卡软件使用，一般也都使用EasyEUICC

二、推荐方案

目前 性价比最高、最推荐 的方案是：

✔ 方案：购买 eSIM 白卡 + 使用开源 EasyEUICC

  • 白卡价格：约 25 元
  • 软件推荐：EasyEUICC（OpenEUICC）
  • 开源地址：
    https://gitea.angry.im/PeterCxy/OpenEUICC

EasyEUICC-1.4.0.apk

优势：

  • 完全开源，透明可信；
  • 支持安卓手机直接写卡；
  • 操作成熟，社区资料丰富；
  • 价格低，适合长期使用。
