箭头的定义

从上面可以看出来，一个箭头可以由6个参数来决定：

  • 箭头的长度 arrowLength (不使用)
  • 箭头的角度 angle
  • 箭头的斜边长 sideLength
  • 箭头的内角度 innerAngle
  • 头部的宽度 headWidth
  • 尾部的宽度 tailWidth

注意：为了保持箭头的形状，一般不成比例的放大缩小箭头，而是根据需要只调整箭头的长度，不改变箭头的形状。成比例缩放箭头效果如下：
